How to build and run using command line
----------------------------------------

Steps:
1) Open command line in the root directory
2) Type 'javac TestCreatues.java'  -- to compile
3) Once it compiles, type java TestCreatures

----------------------------------------
How to build and run jar file
----------------------------------------
Steps:
1) Open command line in the root directory
2) Type 'java -jar HW1.jar'
