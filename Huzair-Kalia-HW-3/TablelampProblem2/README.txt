----------------------------------------
How to build and run using command line
----------------------------------------

Steps:
1) Open command line in the root directory
2) Type 'javac -d "Main" Main\MainClass.java'  -- then press enter
3) Type 'javac -d "Main" -classpath "Main" Main\TableLamp.java'  -- to compile
4) Once it compiles, type java -classpath "Main" Main.MainClass

----------------------------------------
How to build and run jar file
----------------------------------------
Steps:
1) Open command line in the root directory
2) Type 'java -jar TLP2.jar'
